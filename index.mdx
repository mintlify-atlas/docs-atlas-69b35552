## Why KiroGraph?

When you ask Kiro to work on a complex task, it explores your codebase using file reads, grep, and glob searches. Every one of those is a tool call — and tool calls consume context and slow things down.

KiroGraph replaces that entire loop with a pre-built graph stored locally in SQLite. One call to `kirograph_context` returns entry points, related symbols, call edges, and code snippets — often everything the agent needs to understand the task without reading a single file.

## Integrations

KiroGraph has native support for **Kiro** and experimental support for **34 other MCP-capable tools** including Cursor, Claude Code, GitHub Copilot, Windsurf, Cline, Codex, and more.

```bash
kirograph install          # Auto-detect all tools and configure them
kirograph install --target kiro    # Kiro only
kirograph install --target cursor  # Cursor only
kirograph install --all            # All detected platforms, no prompt
```
